# `<gem-book>`

Render markdown into a website

_The current style comes from [Docute](https://github.com/egoist/docute)_

[Documentation](https://gem-book.netlify.com/)

## Development

```bash
# development cli
npm run dev:cli
# watch docs
npm run dev:docs
# development `<gem-book>`
npm run dev:website
# install the cli locally
npm link
```
